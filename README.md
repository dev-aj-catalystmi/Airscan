# airscan-example

Vision framework based lib to scan documents also support OCR

## Installation

```sh
npm install airscan-example

cd ios

pod install
```

## Usage

```js
import Airscan from "airscan-example";

// ...

<Button title="Click here" onPress={() => Airscan.startScanning()} />
```
